# Noah-public
카운터온라인 Noah의 소스코드 입니다
이전의 Noah는 4 vs 4 세션제 MMR 기반 파티 매치메이킹 시스템으로 이루어진 팀 데스매치가 주 컨텐츠였으나
최신 버전에서는 해당 시스템을 제거하고 세션 하나에 중도참여가 가능하도록 변경되었으며, 많은 UI적 패치가 이루어졌습니다
아쉽지만 개발자의 번아웃에 개발이 잠시 중단되었으며, 다시 컨텐츠의 방향성을 찾아가는 도중 안타깝게도 개발 중단 결정을 하게 되었습니다.
1년 반동안의 길고 긴 프로젝트를 여기서 마무리짓습니다.
